# Release Notes for v0.1.3

## swirl-string-core v0.1.3

### Changes
- Updated build configuration for better cross-platform compatibility
- Fixed GitHub Actions workflow for automated wheel building
- Improved C++20 compiler flags for Linux builds

### Supported Platforms
- Windows: Python 3.9, 3.10, 3.11, 3.12, 3.13
- Linux: Python 3.9, 3.10, 3.11, 3.12, 3.13 (wheels via GitHub Actions)
- macOS: Python 3.9, 3.10, 3.11, 3.12, 3.13 (wheels via GitHub Actions)

### Installation
```bash
pip install swirl-string-core
```
